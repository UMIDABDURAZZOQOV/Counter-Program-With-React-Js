# Counter-Program-With-React-Js

📘 React Counter App

A simple and interactive counter application built using React.js. This project demonstrates the use of React components, state management, and event handling.

🚀 Features
➕ Increment the counter
➖ Decrement the counter
🔄 Reset the counter
⚡ Fast and responsive UI
🎯 Built with functional components and React Hooks
🛠️ Technologies Used
React.js
JavaScript (ES6+)
HTML5
CSS3


💡 How It Works
The counter value is stored using React's useState Hook.
Buttons trigger functions to update the state:
Increment increases the value by 1
Decrement decreases the value by 1
Reset sets the value back to 0
React automatically updates the UI when the state changes.
